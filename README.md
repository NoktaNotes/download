
## About Nokta Notes
**Nokta notes** is a free, no nonsense note editor with a clean interface and a focus on usability. It uses its own file format (.nokta files) and supports unlimited nested notes

### Key Features
* Clean editor. Unlimited notes. Drag-and-drop organization
* Rich text and Markdown editor
* Flexible panels: hierarchy, table of contents, editor
* Search and replace
* Dark and light theme
* Always saved, never lose your work
* Lightweight and fast

### System Requirements
- Windows 11
- 64-bit processor

## Support

### Bug Reports & Feedback

Found a bug or have a suggestion? Let me know via GitHub:

1. Go to [GitHub Issues](https://github.com/NoktaNotes/download/issues)
2. Click **New issue**
3. Describe what happened (or what you'd like to see)
4. Click **Submit new issue**

That's it! I'll respond as soon as I can.

### What to Include in a Bug Report

- What you were doing when it happened
- What you expected to happen
- What actually happened
- Your Windows version (if relevant)

Screenshots are always helpful.

### Feature Requests

Have an idea for Nokta? Open an issue and tag it as a suggestion. I can't promise everything will be implemented, but I read every request.
